# libmjolnir
A WebUSB implementation of the Odin protocol, used for interfacing with Samsung devices in "Download mode"

## Status
### Working
- PIT parsing
- Odin handshake
- Download PIT
- Device Reboot
- Erase userdata
- Flash partition

### Planned
- Upload PIT

## Building
`yarn build`

## Unit tests
`yarn test`
